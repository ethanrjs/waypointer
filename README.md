# Waypointer

Fabric mod for Hypixel Skyblock that aims to be the best possible waypoint manager.
Built for Minecraft 1.21.11 (Fabric, client-only).
<img width="1280" height="675" alt="image" src="https://github.com/user-attachments/assets/bb01bb0b-8e2a-4c46-8ffd-57289e7a1f19" />

## Features

- Waypoints organized per Skyblock world type
- Ordered routes
- World rendering
- Extremely compact import/export format
- Importable from chat messages
- Keybindings
- Clean UI for editing and management
- Waypoint skipping in sequence

<img width="1280" height="675" alt="image" src="https://github.com/user-attachments/assets/1acdce6f-5de9-4a1c-8be7-ef1ccb20977d" />

## Waypoint Import Compatibility 

- Coleweight
- Skytils
- Skyblocker
- Soopy
- JSON-based

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.2+
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [owo-lib](https://modrinth.com/mod/owo-lib) 0.13.0+
- *(optional)* Hypixel Mod API 1.0+. Improves zone detection

## Building

```powershell
./gradlew build
```

## Testing

```powershell
./gradlew test
```

## Layout

- `src/main/` — shared logic: codec, config, core waypoint/zone/group types, chat scanners.
- `src/client/` — client-only: rendering, screens, keybinds, location/zone tracking, chat hooks.
- `src/test/` — JUnit 5 tests for codec, scanners, storage, and progression.
